\[\text{Al-Khwarizmi} \rightarrow \text{Al-gorismus} \rightarrow \text{Algorithm}\]

Al-Khwarizmi is also known to be the father of Algebra (Latinization of Al Jabr meaning the Completion ; ‘Al’ means ‘the’ in Arabic). His books “Book of Indian computation” and “Addition and subtraction in Indian arithmetic” also introduced Indian numerals to the Western world. In this books, Al-Khwarizmi introduced procedures that were precise, unambiguous, mechanical, efficient, and correct. In short, they were modern-day “algorithms”.

Example: numbers = [4, 1, 3, 2] returns [1, 2, 3, 4]

def minimum(numbers):
    if len(numbers) == 0:
        return None
    min = numbers[0]
    for item in numbers:
        if item < min:
            min = item
    return min

def sort_ascending(numberss):
    new_list = []
    i = 0
    n = len(numberss)
    while i < n:
        new_list.append(minimum(numberss))
        numberss.remove(minimum(numberss))

    return new_list

Question 5. Factorial \(n!\)#

Factorial of a number \(n\) is the product of all positive integers less than or equal to \(n\) and denoted by that integer and an exclamation point \(!\) i.e.

\[ n! = n~~\times~~(n-1)~~\times~~(n-2) \cdots \times 2 \times 1\]

or

\[ n! = \prod_{i=1}^{n} i\]

For example, \(5!\)

\( \\ = 5 \times 4 \times 3 \times 2 \times 1 \\ = 120\)


Write a function that takes as input an integer \(n\) and returns as output \(n!\) using the formula above.

def factorial(n):
    i = 1
    rnum = 1
    while i < n:
        if i < n:
            rnum *= (i+1)
            i += 1
    return rnum

Test your implementation using the test cases below

Question 6. Fibonacci Sequence#

Al Khwarizmi’s work could not have gained a foothold in the West were it not for the efforts of one man: the 15th century Italian mathematician Leonardo Fibonacci, who saw the potential of the positional Indian numeral system and worked hard to develop it further and propagandize it. But today Fibonacci is most widely known for his famous sequence of numbers known commonly as Fibonacci sequence.


The first two numbers of the Fibonacci sequence are 0 and 1 respectively.

After the first two, each number in the sequence is the sum of the previous two numbers.

The first few numbers of the Fibonacci sequence are 0, 1, 1, 2, 3, 5, 8, …

The value after 8 is 5 + 8 = 13.

The sequence can be summarized as follows:

\(F_0 = 0\)
\(F_1 = 1\)

For all \(n\) > 1,

\(F_n = F_{n-1} + F_{n-2}\)

Write a function that takes as input an integer \(n\) and returns as output the \(𝑛\)th Fibonacci number, using the formula above.

def fibonacci(n):
    fiblist = [0, 1]
    i = 2
    while i <= n:
        fiblist.append(fiblist[i-1] + fiblist[i-2])
        i += 1
    return fiblist[n]
